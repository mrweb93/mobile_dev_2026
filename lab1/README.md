# Лабораторная работа №1
## Создание первого проекта в Android Studio. Запуск на эмуляторе. Обзор интерфейса

**Длительность:** 1 час 30 минут  
**Цель работы:** Ознакомиться со средой разработки Android Studio, создать и запустить простое приложение, изучить структуру проекта и базовые элементы интерфейса.

---

## 1. Теоретическая справка

Android Studio – официальная среда разработки для Android, основанная на IntelliJ IDEA. Она включает в себя всё необходимое для создания приложений: редактор кода, инструменты сборки (Gradle), эмулятор, отладчик и шаблоны проектов.

Проект в Android Studio организован в виде модулей. Основные компоненты:
- **Manifest** – файл `AndroidManifest.xml`, описывающий приложение (имя, иконка, активности, разрешения).
- **Java/Kotlin** – исходный код (активности, классы).
- **Res** – ресурсы: разметки (layout), изображения (drawable), строки (strings.xml), темы (styles.xml).
- **Gradle Scripts** – файлы сборки (`build.gradle`) для настройки зависимостей и параметров компиляции.

Эмулятор Android (AVD – Android Virtual Device) позволяет запускать приложения на виртуальном устройстве без реального телефона.

## 2. Оборудование и программное обеспечение

- Персональный компьютер с ОС Windows / macOS / Linux.
- Установленная Android Studio (скачать с [developer.android.com/studio](https://developer.android.com/studio)).
- Для запуска на реальном устройстве: USB-кабель и включённая отладка по USB (настройки разработчика).

## 3. Порядок выполнения работы

### Этап 1. Создание нового проекта (20 мин)

1. Запустите Android Studio. В окне приветствия выберите **New Project**.
2. В списке шаблонов выберите **Empty Views Activity** (или **Empty Activity** в старых версиях) и нажмите **Next**.
3. Заполните параметры проекта:
   - **Name:** `MyFirstApp` (или любое другое имя).
   - **Package name:** оставьте как предлагается (например, `com.example.myfirstapp`).
   - **Save location:** укажите удобную папку.
   - **Language:** выберите **Kotlin** или **Java** (по указанию преподавателя).
   - **Minimum SDK:** выберите **API 24: Android 7.0 (Nougat)** – этого достаточно для покрытия большинства устройств.
   - Если доступны, снимите галочки с **Use legacy android.support libraries** и **Generate lifecycle methods** (для простоты).
4. Нажмите **Finish**. Студия создаст проект и откроет его. Первая сборка может занять некоторое время (Gradle скачивает зависимости).

### Этап 2. Знакомство с интерфейсом Android Studio (15 мин)

Рассмотрите основные окна (если какое-то окно отсутствует, откройте его через меню `View -> Tool Windows`):

- **Project** (слева) – структура файлов проекта. Переключитесь на режим просмотра **Android** (в выпадающем списке сверху).
- **Editor** (центр) – область редактирования кода и разметок. Сейчас открыт файл `MainActivity.kt` (или `.java`).
- **Logcat** (внизу) – окно для просмотра логов работающего приложения.
- **Toolbar** (верхняя панель) – кнопки запуска, отладки, остановки, выбора конфигурации.

Найдите в структуре проекта (вкладка Project → Android) следующие элементы:
- `app/manifests/AndroidManifest.xml`
- `app/java/com.example.myfirstapp/MainActivity`
- `app/res/layout/activity_main.xml`
- `app/res/values/strings.xml`
- `app/build.gradle` (модуль app)

### Этап 3. Создание виртуального устройства (AVD) (15 мин)

Если у вас есть реальное Android-устройство с отладкой по USB, можете пропустить этот этап и сразу перейти к запуску на устройстве.

1. На панели инструментов нажмите иконку **Device Manager** (или `Tools -> Device Manager`).
2. Нажмите **Create device**.
3. Выберите категорию **Phone** и модель, например **Pixel 6**. Нажмите **Next**.
4. Выберите образ системы (System Image). Рекомендуется скачать образ с API, соответствующим вашему min SDK (например, API 34). Если образа нет, нажмите **Download** рядом с нужной версией.
5. После скачивания выберите образ и нажмите **Next**.
6. Проверьте параметры устройства (можно оставить по умолчанию) и нажмите **Finish**.
7. В Device Manager появится созданное устройство. Нажмите на него правой кнопкой и выберите **Launch** для проверки (эмулятор запустится отдельно). Можно сразу закрыть, он нам понадобится позже.

### Этап 4. Запуск приложения (20 мин)

**На эмуляторе:**
1. В выпадающем списке конфигураций запуска (рядом с зелёным треугольником) убедитесь, что выбрано ваше виртуальное устройство.
2. Нажмите кнопку **Run** (зелёный треугольник) или `Shift+F10`.
3. Дождитесь сборки проекта и запуска эмулятора (если он ещё не запущен). На экране эмулятора появится приложение с надписью **Hello World!**.

**На реальном устройстве:**
1. Включите на телефоне **Режим разработчика** (зайдите в «Настройки» → «О телефоне» → нажмите 7 раз на «Номер сборки»).
2. Зайдите в «Настройки» → «Для разработчиков» → включите **Отладку по USB**.
3. Подключите телефон к компьютеру USB-кабелем. При необходимости подтвердите отладку на телефоне.
4. В Android Studio выберите ваше устройство в списке доступных устройств и нажмите **Run**.

### Этап 5. Модификация приложения (15 мин)

Теперь изменим текст, который отображается на экране.

1. Откройте файл `app/res/layout/activity_main.xml`. Вы увидите разметку в режиме Design (визуальный редактор) или Code (текстовый XML). Переключитесь в режим **Code** (вкладка внизу).
2. Найдите элемент `TextView` с текстом `"Hello World!"`. Обычно он выглядит так:
   ```xml
   <TextView
       android:layout_width="wrap_content"
       android:layout_height="wrap_content"
       android:text="Hello World!"
       ... />
   ```
3. Вместо прямого текста принято использовать ресурсы строк. Создадим строковую переменную:
   - Откройте файл `app/res/values/strings.xml`.
   - Добавьте строку:
     ```xml
     <string name="greeting">Привет, Android!</string>
     ```
4. Вернитесь в `activity_main.xml` и замените `android:text="Hello World!"` на `android:text="@string/greeting"`.
5. Запустите приложение снова. На экране должно появиться «Привет, Android!».

## 4. Индивидуальное задание (вариативно)

Выберите одно из заданий для закрепления материала:
1. Измените цвет текста на синий. Подсказка: добавьте атрибут `android:textColor="#0000FF"` в TextView.
2. Измените размер текста на 24sp. Подсказка: используйте `android:textSize="24sp"`.
3. Добавьте вторую строку (TextView) под существующей с другим текстом. Подсказка: скопируйте существующий TextView и вставьте после него, измените текст и, возможно, добавьте отступ сверху `android:layout_marginTop="16dp"`.

## Возможные ошибки и их решение

- **Gradle sync failed:** Проверьте интернет-соединение, нажмите `File -> Sync Project with Gradle Files`.
- **Emulator не запускается:** Убедитесь, что в BIOS включена виртуализация (Intel VT-x / AMD-V). Для Windows может потребоваться установка HAXM.
- **Device не определяется:** Установите драйвер для вашего телефона (для Windows). Проверьте, включена ли отладка по USB.
- **Приложение не запускается на устройстве с ошибкой "INSTALL_FAILED_UPDATE_INCOMPATIBLE":** Удалите старую версию приложения с телефона.
